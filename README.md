# M4B Suite

Android-застосунок для роботи з аудіокнигами: створення M4B з MP3, редагування
готових M4B, завантаження аудіо з YouTube.

## Структура проєкту

```
m4b_suite/
├── main.py                    # Точка входу, навігація (3 вкладки)
├── buildozer.spec             # Конфігурація збірки APK
├── screens/
│   ├── create_m4b.py          # Інструмент 1: Створити M4B
│   ├── edit_m4b.py            # Інструмент 2: Редагувати M4B
│   └── youtube_downloader.py  # Інструмент 3: YouTube Downloader
├── common/
│   └── file_manager.py        # Спільний файловий менеджер
└── .github/workflows/build.yml # Автозбірка APK через GitHub Actions
```

## Як зібрати APK (рекомендований спосіб — GitHub Actions)

1. Створи новий репозиторій на GitHub
2. Заливаєш весь цей проєкт:
   ```
   cd m4b_suite
   git init
   git add .
   git commit -m "Початковий каркас застосунку"
   git remote add origin https://github.com/ТВІЙ_НІК/m4b-suite.git
   git push -u origin main
   ```
3. GitHub Actions автоматично запустить збірку (вкладка "Actions" у репозиторії)
4. Через 15-30 хв готовий APK можна скачати у розділі "Artifacts" завершеного workflow

## Локальний запуск для тестування логіки (без Android, на десктопі/Termux)

```
pip install kivy kivymd
python main.py
```

⚠️ Файловий менеджер і деякі функції (доступ до сховища) працюють коректно
лише на реальному Android-пристрої.
